# BOJ-ADDON-google-extension
## 백준 에드온이란?
### 요약
백준 사이트 '문제 페이지'에서 문제, 입력, 출력, 입력 예시, 출력 예시 값을 저장하여 '제출 페이지'에서 볼 수 있게 해준다.
